簡単に漢字テストを Let's make a Kanji test easily

このサイトでは、学習年からランダムに問題を作る機能や、漢字を一つずつ選んで作る機能があり、自分で問題文を作らなくても簡単に漢字テストを作ることができます。 This site has a function to randomly create questions from the year of study and a function to select and create Kanji one by one, so you can easily create a Kanji test without creating your own question sentences.
